# Distance-Converter
# KM ↔ Miles Converter

A simple, clean distance unit converter built with HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just open the file and go.

## What It Does

- Enter any number and instantly convert between **kilometers and miles**
- Toggle the direction: **km → miles** or **miles → km**
- Results update in real time as you type

## How to Use

1. Download `unit-converter.html`
2. Open it in any web browser (Chrome, Firefox, Safari, Edge)
3. Type a number, pick your direction, read the result

No installation. No internet connection required.

## The Math

| From | To | Formula |
|------|----|---------|
| Kilometers | Miles | km × 0.621371 |
| Miles | Kilometers | mi × 1.60934 |
